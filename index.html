Projects

Ignite Pickup Boxes System

Upgraded and improved an existing smart locker pickup system used by the Clarkson Ignite Makerspace. The platform allows students to receive automated email notifications with QR codes and retrieve their 3D prints from secure locker boxes controlled by Raspberry Pi and Arduino-based hardware.

I migrated the system from a Raspberry Pi 3 to a Raspberry Pi 5 and improved from a platform that previously required multiple daily reboots into a stable continuously running system suitable for real operational use.

Key Issues Addressed

  • Upgraded system hardware from Raspberry Pi 3 to Raspberry Pi 5
  • Improved reliability to support continuous operation without frequent reboots
  • Debugged and improved backend services built with Python and Flask
  • Made frontend codebase changes to improve usability and system behavior
  • Troubleshot hardware interaction between Raspberry Pi and Arduino controllers
  • Supported Docker-based deployment and operational environment improvements
  • Maintained database and email notification workflows used in production

Tech Stack: Raspberry Pi 5, Python, Flask, PostgreSQL, Docker, Arduino, SMTP

RFID Student Sign-In System

Leading a team building a smart room sign-in system where students tap their ID cards on an RFID scanner connected to a Raspberry Pi. First-time users register through a UI and are stored in PostgreSQL, while returning users are validated and their visit timestamps are logged to Google Sheets.

Key Contributions

  • Helping define the backend structure and project workflow for team development
  • Designing a system that supports first-time registration and returning-user validation
  • Integrating RFID-based identification with Raspberry Pi and database storage
  • Planning visit logging workflows for external reporting through Google Sheets
  • Coordinating a team-based software engineering project from requirements into implementation

Tech Stack: Raspberry Pi, Python, RFID, Flask, PostgreSQL, Google Sheets API
